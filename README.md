# NETFISH (Video Streaming Platform) - Schoolproject

NETFISH is een eenvoudige video streaming platform als schoolopdracht.
Het project wordt gemaakt met **PHP** en **MySQL** via **XAMPP**.  
De frontend is al opgezet met een video-grid.

## Technieken

- PHP
- MySQL (Week 2)
- XAMPP (Apache + MySQL)
- Git/GitHub
- Trello (planning)

## Starten (XAMPP)

1. Plaats de projectmap in: `xampp/htdocs/`
2. Start **Apache** (en later MySQL) in het XAMPP Control Panel
3. Open in de browser:
   - `http://localhost/video-streaming-platform/`
